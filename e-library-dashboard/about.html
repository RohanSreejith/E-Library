About E-Library Dashboard

E-Library Dashboard is a modern, user-friendly web application designed for seamless browsing, searching, and filtering of books. It leverages the Google Books API to provide access to a wide range of digital books across genres and authors. Our platform features dynamic book listings, favourites management, user profiles, and both dark and light modes for an optimal reading experience. Built with HTML, CSS, and JavaScript, the E-Library Dashboard is a showcase of responsive design and interactive web technologies.
